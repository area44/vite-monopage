# Vite Monopage

Welcome to the ultimate starter for modern web documentation. This site is powered by **Vite 8**, **MDX**, and **Tailwind CSS 4**.

## Interactive Features

### Dark Mode

Try the theme toggle in the header! It persists your preference using `localStorage` and respects system settings by default.

### Copy to Clipboard

Hover over any code block to see the copy button. It works on both multi-line blocks and individual snippets.

## Markdown Examples

You can use **bold**, _italic_, ~~strikethrough~~, and `inline code` easily. Links are automatically enhanced with an external icon if they point to another domain: [Vite 8 Docs](https://vitejs.dev).

## Custom Components

### Steps Guide

Use the `Steps` and `Step` components to create beautiful ordered guides.

<Steps>
  <Step title="Initialize Project">
    Clone the repository and install dependencies using pnpm. ```bash pnpm install ```
  </Step>
  <Step title="Start Development">
    Run the dev server to see your changes in real-time. ```bash pnpm dev ```
  </Step>
  <Step title="Deploy to Production">
    Build the static site and deploy it to your favorite host. ```bash pnpm build ```
  </Step>
</Steps>

## GFM Style Alerts

We support GitHub Flavored Markdown alerts natively via the `blockquote` component.

> [!NOTE]
> This is a note alert. Useful for general information.

> [!TIP]
> This is a tip alert. Perfect for helpful advice.

> [!IMPORTANT]
> This is an important alert. Key information the user shouldn't miss.

> [!WARNING]
> This is a warning alert. Use this to signal potential issues.

> [!CAUTION]
> This is a caution alert. Critical warnings that require immediate attention.

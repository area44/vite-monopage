## Interactive Features

### Theme Management

The theme toggle is now a floating button in the top-right corner. It persists your preference using `localStorage` and respects system settings by default.

### Syntax Highlighting

We now use **Shiki** for high-quality syntax highlighting. It supports hundreds of languages and themes.

```typescript
interface User {
  id: string;
  name: string;
  email: string;
}

const getUser = async (id: string): Promise<User> => {
  const response = await fetch(`/api/users/${id}`);
  return response.json();
};
```

## Documentation Steps

<Steps>

## GFM Style Alerts

We support GitHub Flavored Markdown alerts natively via the `blockquote` component.

> [!NOTE]
> This is a note alert. Useful for general information.

> [!TIP]
> This is a tip alert. Perfect for helpful advice.

> [!IMPORTANT]
> This is an important alert. Key information the user shouldn't miss.

> [!WARNING]
> This is a warning alert. Use this to signal potential issues.

> [!CAUTION]
> This is a caution alert. Critical warnings that require immediate attention.

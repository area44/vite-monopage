# Vite Monopage

This template provides a minimal setup to get **React**, **Tailwind CSS** working with **Vite**.

## Development

```bash
git clone https://github.com/area44/vite-monopage
cd vite-monopage
pnpm install
pnpm dev
```

Open your web browser and navigate to http://localhost:5173.

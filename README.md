# Vite Monopage

This is a website built with Vite, MDX and Tailwind CSS.

## Development

```bash
git clone https://github.com/area44/vite-monopage
cd vite-monopage
pnpm install
pnpm dev
```

Open your web browser and navigate to http://localhost:5173.
